# Test Report Generator

A web-based application for generating, managing, and exporting student test reports. The application allows teachers to create detailed test reports, track student performance, and generate both individual and aggregate reports.

## Features

- Create and manage student lists
- Generate individual test reports
- Generate monthly and yearly aggregate reports
- Export reports as:
  - PDF documents
  - Image files
  - Printable format
- Mobile-responsive design
- Offline capability with local storage
- Rich statistics and performance tracking

## Technologies Used

- HTML5
- TailwindCSS for styling
- JavaScript (ES6+)
- html2pdf.js for PDF generation
- html2canvas for image export
- Local Storage for data persistence

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/test-report-generator.git
   ```

2. Open `index.html` in your web browser

   or

   Serve it using a local development server:
   ```bash
   npx serve
   ```

## Usage

1. Add students to the roster
2. Create new test reports with student scores
3. View and analyze individual test reports
4. Generate monthly or yearly aggregate reports
5. Export reports in your preferred format (PDF/Image/Print)
